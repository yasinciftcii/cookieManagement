# Cookie Management Plugin
Plugin for permission management within the web page
---------------------------------------------------------------
## Installation

Install my-project with npm

```bash
  npm install https://github.com/yasinciftcii/cookieManagement.git
  cd cookieManagement
  run localhost
```
